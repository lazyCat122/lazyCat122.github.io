vue基本使用

vue的基本使用步骤：

  1. 需要提供标签用于填充数据
  2. 引入vue.js库文件
  3. 可以使用vue的语法做功能了
  4. 把vue提供的数据填充到标签里面

到这个地址下载vue.js库文件：https://vuejs.org/v2/guide/installation.html

实例参数分析

  • el：元素的挂载位置（值可以是CSS选择器或DOM元素）
  • data：模型数据（值是一个对象）

差值表达式用法

  • 将数据填充到HTML标签中
  • 差值表达式支持基本的计算操作。比如：3 undefined---111

vue代码运行原理分析

  • 概述编译过程的概念（vue语法 -> 原生语法）

vue模板语法

前端渲染：把数据填充到HTML标签中

前端渲染方式

  1. 原生js拼接字符串

    将数据以字符串的方式拼接到HTML标签中

  2. 使用前端模板引擎

    art-template、ejs

    渲染完模板后，需要用原生js的方式获取DOM，然后绑定事件

  3. 使用vue特有的模板引擎

数据绑定指令

v-cloak

解决差值表达式中存在的闪动问题

<div v-cloak>{{msg}}</div>

原理：先通过样式隐藏内容，然后再内存中进行值的替换，替换好之后再显示最终的结果

数据响应式

  1. html5中的响应式（屏幕尺寸的变化导致样式的变化）
  2. 数据的响应式（数据的变化导致页面内容的变化）

数据绑定：将数据填充到标签中

v-once 只编译一次：显示内容之后不再具有响应式功能

双向数据绑定

语法:<input type='text' v-model='uname'/>

例子：

mvvm设计思想

m:model v:view vm:view-model (vue)

model: plain js object (JS对象)

view: DOM

model和view通过viewmodel这个中介进行交互

事件绑定

v-on

语法：<input type='button' v-on:click='num++'/>

​ <input type='button @click='num++'/>

事件函数参数传递：

事件对象$event是固定的写法，并且必须作为最后一个参数传递

没有参数，直接使用的函数名handle，可以不用带$event，默认传到handle:function(event){}

普通参数和事件对象：<button v-on:click='say("hi",$event)'>say hi</button>

事件修饰符

<a v-on:click.stop='handle'>跳转</a>

<a v-on:click.prevent="handle">跳转</a>

按键修饰符

.enter回车键：<input v-on:keyup.enter='submit'>

.delete删除键：<input v-on:keyup.delete='handle'>

自定义按键修饰符

Vue.config.keyCodes.f1 = 112

属性绑定

vue如何动态处理属性：v-bind

<a v-bind:href='url'>跳转</a>

缩写形式：<a :href='url'>跳转</a>

样式绑定

class样式处理

isAvtive就是一个标志位

active就是一个类名

对象语法：<div v-bind:class="{ active: isActive }"></div>

通过true和false的方式控制样式显示与否

数组语法：<div v-bind:class="[activeClass, errorClass]"></div>

操作类名，改对应的值就可以了

style样式处理

<div v-bind:style="{color: activeColor, fontSize: fontSize}"></div>

<div v-bind:style="[baseStyle, overridingStyle]"></div>

v-if和v-show的区别

  • v-if控制元素是否渲染到页面
  • v-show控制元素是否显示（已经渲染到页面）

循环结构

  • v-for遍历数组

    <li v-for='item in list'>{{item}}</li>

    <li v-for='(item, index) in list'>{{item}} + '---' + {{index}}</li>

  • key的作用：帮助vue区分不同的元素，从而提高性能

    index为遍历数组的索引。可以把 item.id 改为 index

    <li :key="item.id" v-for='(item, index) in list'>{{item}}+'---'+{{index}}</li>

  • v-for遍历对象

    <div v-for='(value, key, index) in object'></div>

  • v-if和v-for的结合

    <div v-if='value==12' v-for='(value, key, index) in object'></div>
