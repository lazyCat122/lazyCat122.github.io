diary_blog

发表于 2021-11-02

下载代码和软件http://www.lazycat122.com/download/，下载diary_blog.zip和software.zip

  1. 去阿里云买服务器，能获得一个公网IP，有了这个IP地址，别人就能访问了。

    操作系统装Linux的就好，CentOS 7或8都是ok的、然后到我们购买的服务器页面中，配置安全组放通服务器的80端口和mysql端口（自行看阿里云的liunx系统教程）

    然后自己电脑安装Xshell，Xftp，SQLyog

    使用Xshell，Xftp连接服务器：主机填公网ip，用户名填root，密码时创建服务器时设置的密码，端口22

    用xftp在把存放代码的那个文件夹，解压过的(diary_blog)传输到/home目录下

  2. 使用xshell连接服务器，安装mysql和SQLyog远程连接，点击链接看教程。

    https://lazycat122.github.io/2021/10/09/mysql-install/

  3. 使用xshell连接服务器，安装nodejs。依次输入

    wget https://nodejs.org/dist/v14.17.4/node-v14.17.4-linux-x64.tar.xz

    ls （可以看到，多了一个node-v14.17.4-linux-x64.tar.xz文件，这是一个压缩包）

    tar -xvf node-v14.17.4-linux-x64.tar.xz

    mv node-v14.17.4-linux-x64 nodejs

    ln -s /home/nodejs/bin/npm /usr/local/bin/

    ln -s /home/nodejs/bin/node /usr/local/bin/

    然后分别输入如下指令，如果都能显示版本号，证明node.js安装成功：node -v npm -v

  4. 使用SQLyog连接到服务器的mysql后，打开diary_blog文件夹下的diary.sql。这样就相当于把数据添加到了数据库中了。打开user表，可以更改和添加登录的用户名和密码。打开diary表，可以添加和删除数据

  5. 在xshell中输入指令

    cd /home/diary_blog/model/

    vim model.js

    按键盘上的a进入编辑命令，按键盘上的上下左右控制光标，找到host: '47.96.228.137',把47.96.228.137改为你的公网ip，password: '123456789...'中的密码也需要改为你mysql数据库的密码，然后按esc，再按:wq就可以了

  6. 在xshell中输入指令

    cd /home/diary_blog/

    vim index.js

    按键盘上的a进入编辑命令，按键盘上的上下左右控制光标，找到app.listen(3000),把3000改成80

  7. 在xshell中输入指令

  8. 再浏览器上输入你的公网ip就可以访问到了
