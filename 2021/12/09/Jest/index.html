使用Jest修改自动化测试

访问**https://jestjs.io/docs/en/getting-started.html**学习

npm init

npm install jest@24.8.0 -D

修改package.json：”test”: “jest”

运行 npm run test 会自动执行以test.js结尾的文件

Jest简单配置

npx jest --init生成jest.config.js的配置文件

npx jest --coverage

相当于在package.json中的scripts中添加"coverage": "jest --coverage"，然后执行npm run coverage
