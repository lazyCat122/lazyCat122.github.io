    jquery元素操作

    主要是遍历、创建、添加、删除元素操作

    遍历元素

    jquery隐式迭代是对同一类元素做了同样的操作。如果想要给同一类元素做不同操作，就需要用到遍历

    两个语法：

    $("div").each( function(index, domEle) { xxx; } )

    $.each(object, function(index, element) { xxx; } )

    1. each() 方法遍历匹配的每一个元素，主要是用DOM处理。
    2. each每一个里面的回调函数有两个参数：index是每个元素的索引号；demEle是每个DOM元素对象，不是jquery对象
    3. 所以要想使用jquery方法，需要给这个DOM元素转换为jquery对象 $(domEle)

    创建元素

    $("<li></li>"); // 动态创建一个<li>标签

    1. 创建元素

    2. 添加元素

      内部添加( 添加到元素内部 )

      element.append("内容") // 把内容放入匹配元素内部最后面，类似原生appendChild

      外部添加

      element.after("内容") // 把内容放到目标元素后面

      element.before("内容") // 把内容放到目标元素后面

      内部添加元素，生成之后，它们是父子关系

      外部添加元素，生成之后，它们是兄弟关系

    3. 删除元素

      element.remove() // 删除匹配的元素（自身）

      element.empty() // 删除匹配的元素集合中所有的子节点

      element.html() // 清空匹配的元素内容。和empty()效果一样

    jquery尺寸、位置操作

    jquery尺寸

    语法 用法
    width() / height() 取的匹配元素宽度和高度值 只算width / height
    innerWidth() / innerHeight() 取的匹配元素宽度和高度值 包含padding
    outerWidht() / outerHeight() 取的匹配元素宽度和高度值 包含padding、border
    outerWidth(true) outerHeight(true) 取的匹配元素宽度和高度值 包含padding、border、margin

    注意：以上参数为空，则是获取相应值，返回的是数字型

    ​ 如果参数为数字，则是修改相应值，参数可以不用写单位

    jquery位置

    offset() position() scrollTop/scrollLeft()

    1. offset()设置或获取元素偏移

      offset()方法设置或返回被选元素，相对于文档的偏移坐标，跟父级没有关系

    2. position()获取元素偏移

      position()方法用于返回被选元素相对于带有定位的父级偏移坐标，如果父级都没有定位，则以文档为准

    3. scrollTop/scrollLeft()设置或获取元素被卷去的头部和左侧
